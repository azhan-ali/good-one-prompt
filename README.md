# Bad → Good Prompt 🪄

Apna messy, galat spelling wala, Hinglish prompt daal — aur professional English prompt le ja for Windsurf, Cursor, ChatGPT!

## Setup

1. **Groq API key le (FREE hai):**
   - Jaa: https://console.groq.com/keys
   - Sign up kar aur API key generate kar

2. **`.env.local` mein key daal:**
   ```
   GROQ_API_KEY=gsk_your_key_here
   ```

3. **Install aur run:**
   ```bash
   npm install
   npm run dev
   ```

4. **Browser mein khol:** http://localhost:3000

## Tech Stack
- Next.js 14 (App Router)
- Groq AI (Llama 3.1)
- TailwindCSS
- Lucide Icons
- Sonner (Toast notifications)
